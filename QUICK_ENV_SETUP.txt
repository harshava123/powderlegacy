━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. In Supabase Dashboard → SQL Editor
   2. Click "New Query"
   3. Open file: supabase-schema.sql
   4. Copy ENTIRE content
   5. Paste in SQL Editor
   6. Click "Run" button
   7. Wait for success ✅

STEP 6: CREATE STORAGE BUCKETS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. In Supabase Dashboard → Storage
   2. Click "Create bucket"
   
   Bucket 1:
      Name: products
      Public: ✅ YES
      Click "Create"
   
   Bucket 2:
      Name: home
      Public: ✅ YES
      Click "Create"
   
   Bucket 3:
      Name: header
      Public: ✅ YES
      Click "Create"

STEP 7: SET STORAGE POLICIES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   For EACH bucket (products, home, header):
   
   1. Click bucket name
   2. Click "Policies" tab
   3. Click "New policy"
   
   Add these 4 policies:
   
   Policy 1 - Public Read:
      • All operations: SELECT
      • Policy expression: true
   
   Policy 2 - Auth Upload:
      • All operations: INSERT
      • Policy expression: (auth.role() = 'authenticated')
   
   Policy 3 - Auth Update:
      • All operations: UPDATE
      • Policy expression: (auth.role() = 'authenticated')
   
   Policy 4 - Auth Delete:
      • All operations: DELETE
      • Policy expression: (auth.role() = 'authenticated')
