🎯 HOW TO USE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ADDING IMAGES:
   → Click "+ Add Image"
   → Enter image URL
   → Press Enter or click "Add Image"

2. DELETING IMAGES:
   → Hover over image
   → Click trash icon 🗑
   → Image removed instantly

3. REORDERING IMAGES:
   → Hover over image
   → Use ↑ or ↓ arrows
   → First image becomes primary

4. CHANGING IMAGES:
   → Update the URL in text field
   → See live preview
   → Save changes
